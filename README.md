\## 📌 About



This repository is my personal log of Codeforces solutions, organized by date. The goal is consistency — solving problems every day, building intuition for common patterns (greedy, math, implementation, DP, etc.), and tracking progress over time.



> 💡 If you're also learning competitive programming, feel free to fork this, follow along, or suggest improvements via issues/PRs.
